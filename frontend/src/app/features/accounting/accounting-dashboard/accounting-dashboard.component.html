info À propos de l'export Sage

Ce module génère un fichier d'écritures comptables structuré selon le format Sage PNM/CSV. Il inclut automatiquement :

  • check_circle Journal des Ventes (VT) : Invoices validées et payées.
  • check_circle Journal de Trésorerie (BQ/CA) : Règlements clients par banque ou caisse.
  • check_circle Journal des Achats (AC) : Dépenses et factures fournisseurs validées.

warning Standard International Adaptatif

Le Plan Comptable utilisé est basé sur les standards internationaux (ex: 1100 pour Clients, 4100 pour Ventes). Si vous avez besoin d'ajuster ces codes pour un standard national spécifique, veuillez contacter l'administrateur.
