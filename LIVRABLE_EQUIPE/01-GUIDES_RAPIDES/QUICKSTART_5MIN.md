## 🎯 C'EST QUOI OPTISAAS?

Logiciel pour gestion de **magasins optiques** (lunettes).

**Ce qu'il fait**:
- Gère clients (profils + historique)
- Crée devis → factures → reçoit paiements
- Gère stock (montures, verres, lentilles)
- Donne points fidélité clients
- Paie automatisée employés
- Caisse quotidienne
- Comptabilité Maroc

**Tech**: Angular / NestJS / PostgreSQL / Docker

## 🔄 LE CYCLE (Moment d'un client)

```
1. Client arrive → Vendeur crée FICHE (optique)
2. Vendeur choisit produits → DEVIS généré
3. Client accepte → FACTURE validée
4. Client paie → Points fidélité gagnés
5. Fournisseur expédie → Stock reçu
6. Vendeur confectionne → Client reprend lunettes
7. Dossier CLÔTURÉ

Total: 2-7 jours typiquement
```
