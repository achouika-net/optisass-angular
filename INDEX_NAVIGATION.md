## 📝 NOTES IMPORTANTES

1. **Tous les modules sont mutuellement dépendants** - Il n'y a pas de module isolé
2. **Stock est critique** - Vérification avant DEVIS→VALIDEE obligatoire
3. **Points fidélité automatiques** - Déclenché facture PAYEE
4. **Commissions vendeur auto-calculées** - Mensuel, sur factures payées
5. **Multi-tenant stricte** - Isolation par centreId obligatoire
6. **Audit complet** - Toutes modifications tracées (qui, quand, avant/après)
7. **Validations en cascade** - Niveau application + base de données
8. **Transactions atomiques** - Surtout facturation + paiements + stock
